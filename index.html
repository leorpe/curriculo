Projetos

  • Automação de Builds

    Realizemos a automação de builds, que eram gerados via linha de comando, com o uso do Jenkins. Agora os builds passam por uma Pipeline de Build que facilita e agiliza a entrega de software

  • Automação de Alerta de Conflitos

    Com diversos desenvolvedores atuando em projetos e sustentação, vários branches são criados. Estes branches acabam gerando conflitos que precisam ser solucionados antes de serem entregues para instalação em produção (Merge). Para alertar os envolvidos nestes conflitos, criamos uma planilha excell que consulta os registros na base de dados e dispara e-mails avisando os interessados. Semanalmente são enviados em média 650 alertas.

  • Aplicação do SonarQube para Analise dos Códigos

    Para melhorar a qualidade dos códigos gerados pelos desenvolvedores nossa equipe realizou a impantação do SonarQube. O SonarQube foi inserido num dos passos da Pipeline de Build e dependendo da qualidade o build falhava.

  • Implantação de Alertas via Rocket.Chat

    Enviávamos alertas por e-mail para os usuários que realizavam commits e criavam tags para que eles soubessem que os processos de build e deploy foram iniciados por eles. Para centralizar esses alertas, utilizamos o Rochet.Chat. Aqueles que quisessem, poderiam receber os alertas por e-mail também. A ferramenta foi tão bem recebida que os times de desenvolvimento usavam para discutir os projetos e centralização de comunicações.
